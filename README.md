# 🎲 Digitaler Würfel – Blankenhagen Spieleentwicklung

Dies ist das offizielle digitale Würfel-Tool für die Brettspiel-Prototypen von **Blankenhagen Spieleentwicklung** (Plochingen). 

## 📌 Über das Projekt
Dieses Tool dient als barrierefreie und haptik-ergänzende Lösung für unsere Spiele-Prototypen, wie zum Beispiel das Projekt **"Schattenschlucht / Der Letzte Zahlt"**. Da unsere Prototypen oft auf wetterfester Plane gedruckt oder als Marketing-Editionen auf Papier verteilt werden, ermöglicht dieser Web-Würfel einen sofortigen Spieleinstieg per Smartphone.

### Kernfunktionen:
- **Universelle Nutzbarkeit:** Funktioniert für alle Spiele, die einen Standard-W6 benötigen.
- **Optimiert für Mobile:** Schnelle Ladezeiten und einfache Bedienung auf Stadtfesten oder unterwegs.
- **Guerilla-Marketing-Ready:** Verknüpfung via QR-Code auf unseren Print-Editionen.
- **Regionaler Bezug:** Entwickelt für den Einsatz bei lokalen Events in Plochingen und Umgebung.
